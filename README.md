# FPS Sci-Fi Microgame

A modified Unity FPS Microgame project developed as a portfolio project. The project demonstrates gameplay programming, AI behavior, NavMesh navigation, UI systems, audio implementation, and Unity scene development.

## Project Overview

This project is based on Unity's FPS Microgame and has been modified and expanded to explore custom gameplay systems and development workflows.

## Features

* First-person FPS gameplay
* Enemy AI and combat systems
* NavMesh-based enemy navigation
* Objective and mission systems
* Primary and secondary objective UI
* Center-screen objective notifications
* Custom objective notification audio
* Configurable notification audio pitch
* Custom UI behavior and presentation
* Sci-fi environment and lighting
* WebGL build support

## Technologies

* Unity
* C#
* Unity AI Navigation / NavMesh
* Universal Render Pipeline (URP)
* Unity Input System
* Git
* GitHub
* WebGL

## Development

The project is actively developed as a Unity FPS gameplay and systems project. The repository contains the Unity source project and custom modifications made during development.

## Running the Project

1. Clone or download the repository.
2. Open the project using the compatible version of Unity.
3. Open the main game scene.
4. Press Play in the Unity Editor.
